# macage

Find out when your Mac first booted up. The start of serious Macage.

## Getting Started
Install the module with: `npm install macage`

## Examples
```
$ macage
Born On: Nov 16 2012
Age: 1 year, 6 months, 3 weeks (564 days to be exact).
```
